# Gym Tracker

A single-file personal fitness app — workouts, blood pressure, habit streaks, a research-backed weekly plan, nutrition (recipes, meals, macros), shopping list, weigh-ins, and water/sleep tracking. No build step, no dependencies: everything lives in `index.html` and saves to the browser's localStorage.

## Use it

- **On a computer:** double-click `index.html`.
- **On a phone (installable app):** host the folder (GitHub Pages works great), open the URL in your phone's browser, then **Add to Home Screen**. It installs like a native app, works offline, and keeps its own data.

## Data & backups

All data stays in your browser — nothing is sent anywhere. Use **Settings → Data** to export/import a full JSON backup; the app also keeps an automatic daily snapshot.

`HEALTH-RESEARCH.md` holds the evidence review (training volume, protein targets, cardio interference) that seeded the weekly plan and nutrition targets.
